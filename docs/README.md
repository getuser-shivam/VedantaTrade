# VedantaTrade

A comprehensive pharmaceutical distribution management system built with Flutter for the Nepalese market.

## 🚀 Quick Start

### Prerequisites
- Flutter SDK 3.19.0+
- Dart SDK compatible with Flutter
- Android Studio / VS Code with Flutter extensions

### Setup
```bash
# Clone the repository
git clone https://github.com/your-org/vedanta-trade.git
cd vedanta-trade

# Install dependencies
flutter pub get

# Run the app
flutter run
```

## 📱 Features

- **Product Catalog**: Comprehensive pharmaceutical product database
- **Order Management**: Complete order lifecycle tracking
- **Inventory Management**: Real-time stock monitoring
- **Distribution Network**: Multi-tier distribution management
- **Sales Analytics**: Comprehensive dashboard and reporting
- **Marketing Campaigns**: Campaign management and analytics
- **Customer Management**: Complete CRM system
- **Financial Management**: Accounting and reporting
