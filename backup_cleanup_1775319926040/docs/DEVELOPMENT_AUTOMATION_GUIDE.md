## **🛠️ Usage Examples**

### **Complete Development Workflow**
```bash
# Run full development workflow
dart run tools/vedanta_trade_dev_tool.dart full

# Quick analysis only
dart run tools/vedanta_trade_dev_tool.dart analyze

# Quick build only
dart run tools/vedanta_trade_dev_tool.dart build

# Quick git operations
dart run tools/vedanta_trade_dev_tool.dart git
```

### **Continuous Monitoring**
```bash
# Start continuous monitoring
dart run tools/vedanta_trade_automation.dart monitor

# Run comprehensive workflow
dart run tools/vedanta_trade_automation.dart workflow

# Health check
dart run tools/vedanta_trade_automation.dart health

# Generate status report
dart run tools/vedanta_trade_automation.dart status
```

### **GitHub Integration**
```bash
# Initialize GitHub integration
dart run tools/vedanta_trade_github.dart init

# Sync with remote
dart run tools/vedanta_trade_github.dart sync

# Commit changes
dart run tools/vedanta_trade_github.dart commit "feat: Add new feature"

# Create release
dart run tools/vedanta_trade_github.dart release v3.2.2 "New Release"

# Create feature branch
dart run tools/vedanta_trade_github.dart branch user-authentication

# Merge feature branch
dart run tools/vedanta_trade_github.dart merge feature/user-authentication
```
